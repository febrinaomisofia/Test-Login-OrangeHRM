# Test-Login-OrangeHRM
Repository ini berisi hasil test untuk login website OrangeHRM menggunakan Cypress. Dimana pengujian ini menggunakan UI Automation Test. Code untuk test casenya berada pada file login.cy.js

## Test Cases

- TC001 Verify Login Success
- TC002 Verify Dashboard Header Displayed
- TC003 Verify Search Menu Displayed
- TC004 Verify User Dropdown Displayed
- TC005 Verify Admin Menu Displayed
- TC006 Verify PIM Menu Displayed
- TC007 Verify Leave Menu Displayed
- TC008 Verify Time Menu Displayed
- TC009 Verify Recruitment Menu Displayed
- TC010 Verify My Info Menu Displayed
- TC011 Verify User Dropdown Exists
- TC012 Verify Search Input Exists

## Tools
- Cypress
- JavaScript
- Chrome Browser

## Result
All test cases passed successfully.
